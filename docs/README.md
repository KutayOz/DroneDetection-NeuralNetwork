# Teknik Dokumantasyon

Bu klasor gelismis kullanicilar ve gelistiriciler icin teknik referans icermektedir.

**Normal kullanim icin bu dokumanlara bakmak zorunda degilsiniz.** Sadece `setup.bat` veya `./setup.sh` calistirin, GUI sizi yonlendirecektir.

## Icerik

| Dokuman | Aciklama |
|---------|----------|
| [configuration.md](configuration.md) | Tum YAML konfigurasyon parametreleri ve anlamlari |
| [api-reference.md](api-reference.md) | Python API referansi (Pipeline, Config, Track vb.) |

## Ne Zaman Bu Dokumanlara Ihtiyaciniz Olur?

- **configuration.md**: GUI yerine YAML dosyalari ile calisirken, veya tracking/detection parametrelerini ince ayar yaparken
- **api-reference.md**: Python kodu yazarak sistemi kullanirken, veya kendi uygulamaniza entegre ederken
